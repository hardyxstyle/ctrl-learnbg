🔢 Какво е алгоритъм?

АЛГОРИТЪМ
Точна последователност от стъпки за решаване на даден проблем. Характеристики: крайност, определеност, изпълнимост, резултатност.

Видове алгоритмични структури

  • Линейна — стъпките се изпълняват последователно
  • Разклонена (условна) — избор между два пътя (if/else)
  • Циклична — повтаряне на стъпки (for, while)

🔄 Циклични конструкции

For цикъл — за известен брой повторения

for i in range(5):
    print(i)  # Отпечатва 0,1,2,3,4

While цикъл — докато условието е вярно

x = 0
while x < 5:
    print(x)
    x = x + 1
💡 range(n) генерира числа от 0 до n-1. range(a, b) — от a до b-1.

📦 Модули и библиотеки в Python

Готови колекции от функции, които можем да използваме с import.

import math
print(math.sqrt(16))  # 4.0
print(math.pi)         # 3.14159...
  • math — математически функции (sqrt, sin, cos, pi)
  • random — случайни числа
  • turtle — рисуване с костенурка
  • time — работа с дата и час

Задача 2: Прочети кода
Средна
Какво ще отпечата следният код?
for i in range(1, 6):
   print(i * 2)
Ще отпечата: 2, 4, 6, 8, 10 (всяко на нов ред). range(1,6) = 1,2,3,4,5 и умножаваме по 2.
Задача 3: Напиши алгоритъм
Трудна
Напиши Python код, който отпечатва числата от 1 до 100, но за кратните на 3 отпечатва "Fizz", а за кратните на 5 — "Buzz".
for i in range(1, 101):
    if i % 3 == 0:
        print("Fizz")
    elif i % 5 == 0:
        print("Buzz")
    else:
        print(i)
